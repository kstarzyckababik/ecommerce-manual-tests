# UX Observations

## UX-001 – Checkout with empty cart
**Observation:** User can start checkout process with empty cart.  
**Impact:** User confusion, unclear purchase flow.  
**Recommendation:** Block checkout button or display clear message that cart is empty.

---

## UX-002 – Error messages visibility
**Observation:** Validation messages during checkout are short and may be unclear for new users.  
**Impact:** User may not understand what needs to be corrected.  
**Recommendation:** Display more descriptive validation messages near input fields.
